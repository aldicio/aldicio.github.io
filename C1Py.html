__________LIMITES

from sympy import * from sympy import limit x=symbols('x') f = sin(x)/x g = 1/(x-1) limit(f,x,0) limit(g,x,1,'-') limit(g,x,1,'+') limit(g,x,oo) limit(g,x,-oo)
Significado dos comandos acima:
from sympy import * : biblioteca para fazer cálculos. x=symbols('x'): transforma x em uma variável simbólica
f=sin(x)/x: f é a função $\dfrac{sen(x)}{x}$. limit(f,x,a): calcula o limite de $f$ em relação a $x$ no ponto $x=a$, ou seja, $\displaystyle \lim_{x\to a}f(x)$.
limit(g,x,1,'-'): calcula o limite lateral à esquerda em relação a x no ponto x=1, ou seja, $\displaystyle \lim_{x\to 1^{-}}g(x)$.
limit(g,x,1,'+'): calcula o limite lateral à direira em relação a x no ponto x=1.
limit(g,x,oo): calcula o limite quando x tende para mais infinito, ou seja, $\displaystyle \lim_{x\to \infty}g(x)$.
limit(g,x,-oo): calcula o limite quando x tende para menos infinito.
